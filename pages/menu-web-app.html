MENU WEB APP

~~INTRODUCTION~~

This is my coding project for one of my coding modules, it is a menu/ordering app for a restaurant called Delicious Delight that sells tranditional food and snacks. It uses javascript for the code and Xampp & MySQL for the databases. For now, let me show you some features.

~~FUNCTIONS~~

First we have the main page where all the menu items are displayed. This is the page they see when the first click in.

MAIN FUNCTIONS

One main function is adding in a new menu item and it will show up on the main page. The item can be added by entering in the details, Name, Image, Price and Details. For this example I just added a dragonfruit.

Another function is being able to edit already existing menu items. After changing the details, anything that shows it will be changed accordingly. (The first picture shows how the page looks, the next 2 show the before and after, things changed: Price and details)
